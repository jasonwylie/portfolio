X-Wing List Manager

this app manages x-wing 1.0 list. You can create/delete/modify lists for x-wing 1.0. You can research each pilot, upgrade, and ship in the game. You can see related articles (web/pdf) based on relative criteria keywords extracted using python procedures. Also includes a web/pdf/video lookup for strategy and tactics for the game.

Components

  • Microsoft Access Objects - standard forms, queries and subqueries and reports and custom VBA code
  • Python - external scripts used for pdf keyword extraction, future plans to add in-app python funtionality
  • GitHub addin - uses excellent free Version Control add-in for GitHub integration
